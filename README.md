# Kitabi

A simple Python utility library.

## Installation

To use the functions in this library, you can import them into your Python scripts:

```python
from kitabi import reverse_string
```

## Usage

Currently, the following functions are available:

* `reverse_string(s)`: Reverses a string.

### Example

```python
from kitabi import reverse_string

reversed = reverse_string("hello")
print(reversed)  # "olleh"
```

## Running Tests

To run the tests for this project, run the following command:

```bash
python3 test_kitabi.py
```
